## package fetch

[![Go Report Card](https://goreportcard.com/badge/github.com/invertedv/fetch)](https://goreportcard.com/report/github.com/invertedv/fetch)
[![godoc](https://img.shields.io/badge/go.dev-reference-007d9c?logo=go&logoColor=white)](https://pkg.go.dev/mod/github.com/invertedv/fetch?tab=overview)

Package fetch is a lightweight package to load CSV or XLSX files from the web or a local file. 
The output is a slice of string slices.
